Software Duplication

Brief Description

Software Duplication is simply when a creator wants to copy and paste code or software so they can share it.

However there is also Software Piracy which is the major disadvantage when someone illegally copies and distributes software or the source code.

This "business" has caught the attention of many crime groups in a number of countries. The Business Software

Alliance (BSA), reports that about 36% of all software in current use is stolen. Software Duplication is sadly

common but has caused significant loss in revenue for publishers because they are not getting the money or

credit they deserve which in turn results in higher prices for the cosnumer.

Who started this?

Copy and Pasting code has been a tool ever since it was created. However one of the disadvantages is Software Piracy.

Before 1980 one could freely copy computer programs and pirate without any leagal ramifications. If someone

their own money and personal gain. So Software duplication started as a popluation and not by one singular person.

code and sell the same software they didnt make so they get the money and not the person who made it.
